# Dotfiles

## quick install:
  * `wget --no-check-certificate https://raw.githubusercontent.com/Gonzih/dotfiles/master/.init.sh -O - | sh`

## requirements:
  * git
  * vim
  * wget
  * curl
  * bash
  * sh
  * fish
